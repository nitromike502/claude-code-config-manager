## Annotations

**Key Design Decisions:**
1. **User Global First:** Always show at top of destination list (most common use case)
2. **Card-Based Selection:** Each destination is a full card (not dropdown) for better scannability
3. **Scrollable Area:** Max 5-6 cards visible before scrolling (handles 95% of users with <15 projects)
4. **Single Action Button:** Only "Copy Here" triggers copy (not clicking entire card)
5. **Visual Hierarchy:** Clear separation between source (read-only, top) and destinations (interactive, below)

**Accessibility Notes:**
- Tab order: Close button â†’ Destination cards â†’ Copy buttons â†’ Cancel button
- Escape key closes modal
- Enter key on focused destination card activates copy
- ARIA role="dialog" on modal container
- ARIA-live region for status announcements

**Responsive Behavior:**
- Desktop (>768px): 600px wide modal, centered
- Mobile (<768px): Full-screen modal (100vw x 100vh)
